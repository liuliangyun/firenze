# 德州扑克游戏规则

## 使用道具
一副标准扑克牌去掉大小王后的52张牌进行游戏。

## 游戏人数
至少2个玩家。

## 游戏目的
赢取其他玩家的筹码。

## 游戏玩家
+ 庄家：每局游戏都会有一名庄家，当一局牌局结束后，庄家按照顺时针方向下移一位。
+ 小盲注玩家：庄家左手边为小盲注玩家，在牌局开始之前，需要投下小盲注。
+ 大盲注玩家：小盲注左手边为大盲注玩家，在牌局开始之前，需要投下大盲注，一般而言，大盲注是小盲注的2倍。
+ 普通玩家：除庄家、小盲注玩家、大盲注玩家以外的其他玩家。

### Missing Concept：
1.当只有两个玩家时，大盲注玩家也是庄家

## 游戏轮数
确定好小盲注和大盲注后，游戏开始，游戏一共分为4轮：
+ 第一轮，给每个玩家发两张底牌。小盲注玩家和大盲注玩家分别下注后，从大盲注后的第一个人玩家开始，按照顺时针方向依次表态，玩家可以选择跟注、加
注、过牌、弃牌。
+ 第二轮，翻出3张公共牌，由小盲注开始（如果小盲注已弃牌，由后面最近的玩家开始，以此类推），按照顺时针方向依次表态，玩家可以选择跟注、加注、过牌
、弃牌。
+ 第三轮，翻出第4张公共牌，由小盲注开始（如果小盲注已弃牌，由后面最近的玩家开始，以此类推），按照顺时针方向依次表态，玩家可以选择跟注、加注、过
牌、弃牌。
+ 第四轮，翻出第5张公共牌，由小盲注开始（如果小盲注已弃牌，由后面最近的玩家开始，以此类推），按照顺时针方向依次表态，玩家可以选择跟注、加注、过
牌、弃牌。

### Missing concept
1.游戏玩家在游戏中分为三种状态：<font color=#FF0000>弃牌状态、All in状态、其他状态</font>。

2.如果场上出现玩家加注，将顺时针遍历非弃牌状态的玩家进行表态。
```
例1：
Given 玩家A、B、C、D，所有人均为<其他状态>，目前场上最高下注额为4元，
When 玩家A加注到8元，
Then 将顺时针依次遍历B、C、D进行表态。

例2：
Given 玩家A、B、C、D，A为<其他状态>，其他人均为<All in状态>，目前场上最高下注额为4元，
When 玩家A加注到8元，
Then 将顺时针依次遍历B、C、D进行表态。

例3：
Given 玩家A、B、C、D，B为<弃牌状态>，其他人均为<其他状态>，目前场上最高下注额为4元，
When 玩家A加注到8元，
Then 由于出现了加注，将顺时针依次遍历C、D进行表态。
```
3.游戏进入到下一轮的条件为：场上<其他状态>的玩家（即：除去弃牌和All in的其他玩家）需要下到同样注额的筹码。
```
例1：
Given 玩家A、B、C、D，所有人均为<其他状态>，目前场上最高下注额为4元，
When 玩家A加注到8元，玩家B跟注8元，玩家C跟注8元，玩家D跟注8元，
Then 由于场上玩家均下至相同注额，游戏进入到下一轮。

例2：
Given 玩家A、B、C、D，A为<All in状态>，其他人均为<其他状态>，目前场上最高下注额为4元，
When 玩家B加注到8元，玩家C跟注8元，玩家D跟注8元，
Then 由于场上除了All in的A外，其他玩家均下至相同注额，游戏进入到下一轮。

例3：
Given 玩家A、B、C、D，A为<弃牌状态>，其他人均为<其他状态>，目前场上最高下注额为4元，
When 玩家B加注到8元，玩家C跟注8元，玩家D跟注8元，
Then 由于场上除了弃牌的A外，其他玩家均下至相同注额，游戏进入到下一轮。

例4：
Given 玩家A、B、C、D，所有人均为<其他状态>，目前场上最高下注额为4元，
When 玩家A加注到8元，玩家B跟注8元，玩家C跟注8元，玩家D加注到12元，
Then 由于场上<其他状态>玩家未下至相同注额，游戏无法进入到下一轮。
```

## 玩家行为（表态）
+ 跟注：玩家跟随场上的最高投注额下注称作跟注。
+ 加注：玩家增加场上的最高投注额。
+ 过牌：玩家暂不表态，由下一个玩家进行表态。
+ 弃牌：一旦弃牌，该玩家游戏结束。

### Missing Concept：
1.All in：一旦玩家的剩余筹码不足以跟注到场上的最高投注额时，玩家若想继续游戏，只能选择All in，否则弃牌。
```
例1：
Given 玩家A、B、C、D，玩家A持筹码10，场上最高投注额为12，
When 轮到玩家A进行表态，
Then 玩家A可以选择All in或者弃牌。

例2：
Given 玩家A、B、C、D，玩家A持筹码12，场上最高投注额为12，
When 轮到玩家A进行表态，
Then 玩家A可以选择跟注或者弃牌。

例3：
Given 玩家A、B、C、D，玩家A持筹码20，场上最高投注额为12，
When 轮到玩家A进行表态，
Then 玩家A可以选择跟注、加注或者弃牌。
```

2.<All in状态>的玩家可以一直进行游戏，并参与游戏结算。

3.<All in状态>的玩家只能过牌。
```
例1：
Given 玩家A、B、C、D，玩家A为<All in状态>，
When 轮到玩家A进行表态，
Then 玩家A只能选择过牌。
```

4.<其他状态>的玩家可在任意游戏轮次的表态中选择弃牌。
```

When 轮到玩家A进行表态，
Then 玩家A可以弃牌。

When 轮到玩家A进行表态，

```

5.仅当玩家当前的投注额为场上的最高投注额或者为All in状态时，可以过牌。
```
例1：
Given 玩家A、B、C、D，玩家A是<其他状态>，且是场上的最高投注额，
When 轮到玩家A进行表态时，
Then 玩家A可以选择过牌。

例2：
Given 玩家A、B、C、D，玩家A是<其他状态>，但投注额低于场上的最高投注额，

Then 玩家A不可以选择过牌。

例3：

When 轮到玩家A进行表态时，
Then 玩家A可以选择过牌。

例4：
Given 玩家A、B、C、D，玩家A是<All in状态>，但投注额低于场上的最高投注额，
When 轮到玩家A进行表态时，
Then 玩家A可以选择过牌。
```


## 游戏结算
经过前面四轮，剩余的玩家开始亮牌比大小，玩家用自己的2张底牌和5张公共牌结合在一起，选出5张牌，凑成最大的成牌，成牌最大的玩家为最终赢家，赢得相应
数量的筹码。

### Missing concept
1.游戏结束时，对于输家，如果玩家的投注额大于赢家的投注额，那么输掉的金额为赢家的投注额，否则输掉的金额为所有的投注额；对于赢家，将赢得其
他所有玩家输的金额总和。
```
例1：
Given 玩家A、B、C、D，最终赢家是A，A为<All in状态>，投注额为20；B为<弃牌状态>，投注额为15；C为<All in状态>，投注额为25；D为<其他状态>，
投注额为40，
When 进行游戏结算，
Then B输掉15，C输掉20，D输掉20，A赢取15 + 20 + 20 = 55。

例2：
Given 玩家A、B、C、D，最终赢家是A，A为<All in状态>，投注额为20；B为<弃牌状态>，投注额为30；C为<All in状态>，投注额为18；D为<其他状态>，
投注额为40，
When 进行游戏结算，
Then B输掉20，C输掉18，D输掉20，A赢取20 + 18 + 20 = 58。

例3：
Given 玩家A、B、C、D，最终赢家是A，A为<其他状态>，投注额为40；B为<弃牌状态>，投注额为15；C为<All in状态>，投注额为25；D为<其他状态>，
投注额为40，
When 进行游戏结算，
Then B输掉15，C输掉25，D输掉40，A赢取15 + 25 + 40 = 80。
```

## 比牌方法
比牌先比牌型，大的牌型大于小的牌型，牌型一般分为10种，从大到小为：
+ 皇家同花顺（royal flush）：由AKQJ10五张组成，并且这5张牌花色相同
+ 同花顺(straight flush）：由五张连张同花色的牌组成
+ 4条（four of a kind）：4张同点值的牌加上一张其他任何牌
+ 满堂红（full house）（又称“葫芦”）：3张同点值加上另外一对
+ 同花（flush）：5张牌花色相同，但是不成顺子
+ 顺子（straight）：五张牌连张，至少一张花色不同
+ 3条（three of a kind）：三张牌点值相同，其他两张各异
+ 两对（two pairs）：两对加上一个杂牌
+ 一对（one pair）：一对加上3张杂牌
+ 高牌（high card）：不符合上面任何一种牌型的牌型，由单牌且不连续不同花的组成
