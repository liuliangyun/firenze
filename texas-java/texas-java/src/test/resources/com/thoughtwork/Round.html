德州扑克轮次内加注规则

德州扑克轮次内投注的规则如下：

  • 每一轮次中，尚在活跃玩家依次行动
  • 玩家可采取的行动有：pass，bet，raise，fold，其中：
    • pass表示当前轮空，将行动权交给下一个玩家，只有在没有玩家采取任何行动时，才能pass；
    • bet表示投注，在没有加注的情况下，投注为最小投注金额。如果有加注，那么则为投注额与已投注金额的差值；
    • raise表示加注，一旦加注完成，那么所有未达到加注金额的玩家都需要重新采取行动以补足差额；
    • fold表示放弃，如果任意时刻，玩家选择fold，则退出游戏；
  • 知道所有玩家都投注，且仍在活跃的玩家投注金额相等时，进入下一轮
  • 只剩一位活跃玩家，则游戏结束

下面让我们看看具体的例子，以帮助我们更具体的理解规则。

例如，我们共有 A,B,C三位玩家参与 游戏， 那么第一个采取行动的玩家应该是A， 如果他投注， 则当前底池金额应是最小投注额， 后面所有玩家都应投注最小投注额。 如果B, C依次跟投， 那么最后所有人投注金额相同， 进入下一轮次。
